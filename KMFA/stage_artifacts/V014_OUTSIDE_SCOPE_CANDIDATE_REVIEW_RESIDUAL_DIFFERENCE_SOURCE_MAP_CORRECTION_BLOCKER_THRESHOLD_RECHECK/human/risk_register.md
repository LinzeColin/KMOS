# Risk Register

- Risk: the second blocker observation is mistaken for a blocked-goal threshold.
- Control: this phase keeps the threshold false and does not mark the goal blocked.
- Risk: valid diagnostic responses are mistaken for source-map correction authority.
- Control: every downstream gate remains closed until source-map correction or authoritative value resolution is supplied.
- Risk: private response details leak into public evidence.
- Control: public artifacts contain aggregate counts only and private threshold outputs remain ignored.
