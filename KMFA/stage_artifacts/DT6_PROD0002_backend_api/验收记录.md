## 三、实测抓到的真 bug（单测漏、真起服务才现形）

`GET /api/事实/{名}` 原用**非 ASCII 路径参数名**。Starlette 将其编为正则命名捕获组，非 ASCII 名导致该路由**在真实 HTTP 下永不匹配**——实测返回 Starlette 路由级 `{"detail":"Not Found"}`（非处理器的「无此事实文件」）。**而 `TestClient` 却能通过**，单测全绿。

改为 ASCII `{name}` 后实测通过。**这正是本 goal「禁止用 pytest 绿冒充功能做完」所针对的缺陷类型**——已在代码注释与本档留证。
