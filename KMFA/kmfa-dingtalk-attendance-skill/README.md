# KMFA DingTalk Attendance Skill Package

This folder packages the public-safe operating instructions, configuration templates, and validation checks for KMFA S19 DingTalk attendance automation.

It intentionally excludes private runtime data, DingTalk credentials, DWS resolved identifiers, SQLite, raw OneDrive archives, and report bodies.

Start with `SKILL.md`, then read `references/runbook.md` and `references/configuration.md`.
