# 模型参数文件

## 执行参数

| 参数 | 当前值 | 说明 |
| --- | --- | --- |
| `automation_name` | `每日钉钉DWS归档` | Codex automation 名称 |
| `automation_time_local` | `11:00,19:00` | 每天 11:00 和 19:00 本地时间 |
| `run_source` | `codex_manual` / `codex_automation` | 区分手动或 automation |
| `page_size` | `500` | DWS 单次请求大小，不是停止条件 |
| `full_depth_no_time_or_page_truncation` | `true` | 全时、全盘、全量、深度完整收集，不按页数或时间窗截断 |
| `heartbeat_resource_interval` | `25` | 每处理 25 个资源刷新一次 heartbeat |
| `stale_heartbeat_minutes_before_blocked` | `30` | 无 heartbeat/日志/子进程进展超过该分钟数才判阻塞 |
| `dws_http_timeout_seconds` | `120` | 传给 DWS CLI 的 HTTP 超时秒数 |
| `dws_command_retries` | `3` | DWS 瞬态网络错误最大重试次数 |
| `dws_retry_backoff_seconds` | `5` | DWS 重试退避基准秒数 |
| `target_change_updates_automation_required` | `true` | 增删改目标群后必须同步检查 automation |
| `project_auto_complete_idle_days` | `7` | 项目群连续无新增消息/文件达到该天数后自动完工并转 `manual_only` |
| `hot_storage_days` | `60` | 当前交付包热窗口 |
| `max_failure_attempts` | `3` | 资源主动重试上限 |
| `staging_ttl_hours` | `24` | staging 清理阈值 |
| `log_retention_days` | `15` | 本机日志保留天数 |
| `recent_similarity_days` | `30` | 相似/重名文件辅助窗口 |

## 主要状态字段

- `group_cursors.last_success_cursor_time`
- `group_cursors.last_message_time_seen`
- `config.groups[].completed_at` / `completed_date`：项目群自动或手动完工时间；自动完工时等于最后一次有消息的时间。
- `manifest.storage_tier`
- `manifest.cold_archive_path`：历史字段名保留，当前语义为冷存储文件绝对路径，不是 zip 路径。
- `manifest.cold_archive_inner_path`：历史字段名保留，当前语义为冷存储相对路径，如 `<群名>/files/MM/<文件>`。
- `manifest.attempt_count`
- `manifest.exhausted`
- `manifest.next_action`
- `reports/current_run_heartbeat.json`
