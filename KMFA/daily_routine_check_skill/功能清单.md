# Dingtalk-routine-check｜功能清单

## 只新增一个 automation/skill

名称：`Dingtalk-routine-check`

中文名：`钉钉工作检查`

这个 automation 不负责创建或管理 DWS 采集。它只读取已有 OneDrive 输入，主输入为 zip：

```text
/Users/linzezhang/Library/CloudStorage/OneDrive-Personal/DWS_Outputs.zip
```

这是唯一上游输入。磁盘 `DWS_Outputs/` 目录不存在是正常状态，automation
不得探测、创建、materialize、复制、解压或 fallback 到该目录。ZIP 内部
`DWS_Outputs/<群>/...` 只表示 member 路径。

## 功能 1：读取 DWS 输出

- 读取两个群的 `manifest.csv`。
- 读取两个群的 `chat_records.csv/jsonl`。
- 读取已下载图片/文件路径。
- 统一 message、file、sha256、sender、time、group。
- 不重新扫描钉钉。
- 不下载新图片。
- 从 `DWS_Outputs.zip` 流式读取目标 CSV，不解压大包到本机；OneDrive zip 占位或损坏时 healthcheck 输出 `ZIP_INPUT_UNREADABLE` 和启用条件。
- 不复制 ZIP 或 member 到本机缓存，不在每次运行后自动 evict ZIP；hydration 生命周期由 OneDrive/用户管理。

## 功能 2：固定工作检查

- 每日、每周、每月第三周周五规则检查。
- 支持单人发送规则。
- 支持多人任选其一规则。
- 支持图片、文件、文本消息。
- 到截止时间仍未发，提醒张霖泽。
- 迟发、低置信度、发错群、疑似发错表，提醒张霖泽。

## 功能 3：资金账户明细表 OCR 监控

- 仅监控付款请示群、杨婷。
- 识别 `资金账户明细表`。
- 从 DWS 消息文本按 `cash_monitor.public.yaml` 的配置化金额锚点提取总可用现金；图片/附件候选但无法提取时进入 `NEEDS_REVIEW`，不伪造 OCR 结果。
- 低于硬阈值 50 万：P0 红色。
- 低于软阈值 100 万：P1 黄色。
- 正常：P2 绿色，只记录不打扰。

## 功能 4：资金流水明细 OCR 结构化

- `资金流水明细` 和 `资金明细` 视为同一类。
- 与 `资金账户明细表` 独立检查。
- 提取日期、事由、收付款人、类别、收入、支出、银行、备注。
- 目前不做大额支出异常。
- 目前不接已审批未付款金额。

## 功能 5：日志与数据库

- 不生成汇总 Excel。
- 写 SQLite 私有 ledger。
- 写 OneDrive JSONL 日志。
- 写 notification log。
- 写 OCR job/extraction log。
- 写 `routine_check_results`、`cash_risk_results`、`cash_account_snapshots`、`notification_events`、`data_quality_issues` 明细表。
- 写数据质量问题日志。
- 后续 KPI/工作数据分析可复用这些结构化事实。

## 功能 6：本机存储与清理

- 活跃 SQLite、通知本地配置、JSONL、OCR 缓存、cleanup report 放 OneDrive private runtime/output。
- repo 内 `KMFA/metadata/daily_routine_check/private_runtime/` 只保留占位文件，避免 validator 被运行态文件污染。
- 支持 `--cleanup --dry-run` 与 `--cleanup --apply`；apply 只做 SQLite WAL checkpoint、VACUUM 和 `cleanup_events` 记录。
- cleanup 仅为手工 maintenance，不进入每日 scheduled command。
- 不删除或自动 evict DWS 原始 ZIP。

## 功能 7：GitHub 实时同步治理

- 所有 skill/code/metadata/tests 改动必须在 main 分支同步。
- 不开 branch。
- 不开 PR。
- push 前跑 validator/tests。
- 失败则不 push。
- 提供 `git_autosync.py`，供 Codex Desktop automation 调用。
