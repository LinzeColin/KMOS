# 操作流程

> 注：v1.4 草案里"业务流程"、"操作逻辑与流程"、"操作流程"是同一件事的三个名字。
> 本文件合并为**一个**流程章节。业务规则在 `03_口径字典.md`，本文件只讲怎么走。

## 一、月度主流程

| 步 | 谁 | 做什么 | 产出 |
|---|---|---|---|
| 1 | 你 | 把当月原件放进 `KMFA_MetaData` | 原件 |
| 2 | 系统 | 只读扫描 + hash + 清单 | raw 清单 |
| 3 | 系统 | 抽取 → 标准化 → 匹配 → 计算 | 派生数字 |
| 4 | 系统 | 对 A0 权威基准做零差异校验 | 差异报告 |
| 5 | **你** | 处理待处理队列里的差异 | 裁定记录 |
| 6 | 系统 | 重跑派生链 | 更新数字 |
| 7 | 系统 | 出报告 + 等级 | 报告 |

**差异未闭合 → 报告等级 ≤ B，A 级被阻断。**

## 二、人工裁定流程

```text
差异出现 → 队列 → 你选一个源为准 或 标为"接受差异"
       → 前端只写裁定事件，不改原始数据、不改抽取值
       → 后端重跑
       → 差异消失 或 升级为阻塞
```

## 三、用户手册

## 四、停止条件

发现以下任一情况，**立即停止，不继续**：

1. 原始数据被污染（hash 变了）
2. 金额出现浮点数
3. 0.01 元差异被静默忽略
4. 报告绕过质量等级
5. 缺少可追溯证据
6. **遇到只有 Owner 能解的阻塞** → 写进 `00_我在哪.md` 第二节，本次结束，**禁止新开步骤重审**
