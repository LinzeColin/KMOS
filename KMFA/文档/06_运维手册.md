## 三、排错手册

| 现象 | 原因 | 怎么办 |
|---|---|---|
| 渲染 FAIL：超行数上限 | 机器平面塞太多进人类平面 | **去精简机器平面，不要调大上限** |
| 渲染 FAIL：中文门 | 正文出现未登记的英文术语 | 在 `03_口径字典.md` 第六节加中文条目 |
| 渲染 FAIL：功能清单出现 phase/gate 字样 | 又在往功能清单里写日志 | 挪到 `05_执行与验收.md` |
| `check_blocker_stop` FAIL | 同一 Owner 阻塞被第 2 次重审 | **停。** 阻塞上浮 `00_我在哪.md`，本次结束 |
| 报告等级恒为 D | `lineage_complete` 为 false | 查源优先级链第 1 环有没有接通 |
| 零差异校验无法运行 | 无 A0 权威基准 | 解 `BLK-001` |
| 原件 hash 变了 | **原始数据被污染** | 立即停。从备份恢复原件，查谁写的 |
