# 功能清单

1. 读取经营管理分析任务包 Markdown，固化输入、口径、Excel、PDF、验收、防回退规则。
2. 读取 `config/v6_spec.json`，校验 v6 Sheet 顺序、标签颜色、图表、禁用词和资金公式。
3. 管理 7 个用户输入槽位，并映射到任务包的业务输入组。
4. 为每个输入文件计算 SHA256、大小、扩展名、是否 symlink、可读取 sheet 清单。
5. 生成 public-safe raw index，不把原始敏感 Excel 明文提交到 GitHub。
6. 生成 run manifest，记录期间、输入槽位、输出文件、验收状态、清理状态。
7. 生成 backup registry，登记本期备份状态、GitHub metadata 路径、禁止提交项。
8. 生成数据库 schema 和 SQL 导出，用于长期审计和跨期查询。
9. 运行正式 Excel/PDF 验收脚本，失败时阻断正式交付。
10. 检查正式输出文件命名：`经营管理分析报表 YYYYMM.xlsx`、`董事会经营分析摘要 YYYYMM.pdf`。
11. 检查 PDF 是否来自同期间、已验收通过的 Excel。
12. 记录本地保留策略：最终报告可留在本机，临时缓存、运行日志和数据库必须可清理。
13. 提供非专业人员可读的运行手册、数据合同、规则清单、故障处理方式。
14. 提供 package validator，检查 skill 文件、metadata 结构和脚本可执行性。
15. 提供长期运行入口，后续可接入自动化调度，但默认不触发外部业务动作。
