# 01_模板基准规范｜v6 固化版

## 1. 输出命名

所有正式交付文件必须使用 `FileName YYYYMM` 格式：

| 类型 | 文件名 |
|---|---|
| 管理 Excel | `经营管理分析报表 YYYYMM.xlsx` |
| 董事会 PDF | `董事会经营分析摘要 YYYYMM.pdf` |
| Agent 内部验收报告 | `自动验收报告 YYYYMM.json` |

禁止：

- `YYYY-MM-DD`
- `v7/v8` 出现在正式交付文件名中
- URL 编码文件名
- 乱码文件名
- 额外副本，如 `final_final`、`new`、`copy`

| 05 | `05_客户市场` | 客户质量、行业结构、回款/开票贡献 | `#0891B2` |

## 3. 全局视觉规范

| 项目 | 标准 |
|---|---|
| 字体 | Microsoft YaHei；如环境缺失，使用微软雅黑兼容字体；禁止宋体/Calibri 混杂 |
| 首页标题 | 18pt，白字，深色背景，居中 |
| 其他 Sheet 标题 | 14–16pt，白字，深色背景，居中 |
| 表头 | 11–12pt，加粗，白字，深绿色/深蓝色/深红色背景 |
| 正文 | 10–11pt，水平居中、垂直居中、自动换行 |
| 行高 | 必须保证内容完整显示；标题 28–36，表头 24–30，正文 22–36，长文本 40–72 |
| 列宽 | 必须保证关键文字完整显示；文本列 16–34，金额列 12–18，说明列 30–50 |
| 金额单位 | 首页和管理汇总页默认万元；源表复刻页可保留源表单位 |
| 百分比 | 全局涉及比例必须显示两位小数百分比，例如 `22.53%` |
| 边框 | 浅灰蓝 `#CBD5E1` 细边框 |
| 对齐 | 所有可见区域必须水平居中、垂直居中；长文本自动换行 |

## 4. 禁用词和禁用输出

可见 Excel 和 PDF 中禁止出现：

- `Series1`、`Series2`、`Series 1`、`Series 2`
- `责任角色`
- `CxO`、`CEO`、`CFO`、`COO`、`CMO`、`CTO`
- `Source_Catalog`
- `源文件状态`
- 测试数据、模拟数据、示例数据
- `#####`
- 乱码、错码、URL 编码文件名

## 5. 图表规范

所有图表必须是 Excel 原生可编辑图表，不得插入截图或静态图片。图表必须有中文业务标题、中文图例，不得使用默认图例。

| Sheet | 必需图表 |
|---|---|
| `00_首页总览` | `月度回款与开票趋势（万元）` |
| `01_回款应收项目` | `月度回款与开票趋势（万元）` |
| `03_2026年销售回款` | `外部营业开票额｜最近三年同期对比（元）`、`销售回款｜最近三年同期对比（元）`、`保证金回款｜最近三年同期对比（元）` |
| `04_2026年资金汇总` | `资金流入、流出与累计净流量（元）`，必须是折线图 |
| `05_客户市场` | `行业2026累计回款` |
| `06_资质人力产能` | `三类人员与特种设备人员缺口（人）`、`组织与产能画像（人）` |
| `07_保证金履约` | `按支付类型未退余额（元）`、`按账龄未退金额（元）` |
| `08_趋势预测` | `未来13周回款预测（万元）` |

## 6. 首页总览固定结构

### 标题区

标题：`经营管理分析报表 YYYY/MM｜首页总览`

副标题：`数据截止日：YYYY-MM-DD｜覆盖 2024-01-03 起｜单位：万元`

数据来源：`数据来源：WPS 武汉开明 2026年回款表，三大项目，2026年保证金，应收帐款合同登记，应收账龄表，生产项目状态表、红圈主合同`

若某一来源缺失，Agent 不得静默忽略，必须在生成前停止并列出缺失来源；若用户确认继续，才可在口径说明中写明限制。

### KPI 卡片

首页第一组 KPI 必须按以下顺序排列。每张卡片三行：

1. 指标名
2. 金额或比例
3. `同比 xx.xx%｜环比 xx.xx%`

| 行区 | 指标 |
|---|---|
| 5–7 行 | `2026累计回款`、`2026累计开票`、`2026开票未回款`、`2026合同未完工` |
| 9–11 行 | `本月回款`、`本月开票`、`本月合同`、`本月未开工` |
| 13–15 行 | `6个月内应收`、`6个月以上1年以内应收`、`3年以上应收`、`保证金未退` |

首页必须保留紧凑的“风险机会行动摘要”，但不得使用“责任角色”列，不得出现 CxO 字样；建议列名为：`事项`、`关键数据`、`建议行为`、`优先级`、`预计影响`。
