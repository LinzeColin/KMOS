## 对 Codex 的强制要求

- 第一轮仍然是 PLAN / READ_ONLY。
- 不能先写 UI，不能先生成报告，不能先接自动接口。
- 但凡进入 UI/报告阶段，必须读取 `20_HTML_UIUX_报告预览/` 下的 HTML 样板。
- 所有金额使用整数分或 Decimal，禁止 float。
- 任意 0.01 元差异进入 mismatch / resolution，不得静默通过。
- 前端只能写控制事件、映射规则、处理意见、审批记录；不能污染原始上传数据。
- 原始上传数据仅在本地私有使用，禁止提交公开 GitHub。
