报告生成原则

报告只呈现经营结论、专业附表与必要口径说明，不暴露系统内部核对流程。

周报网页 + PDF现金/回款清单每周固定提醒银行流水过期则降级
月报网页 + PDF利润、应收、税票、项目附表月结后提醒金蝶报表缺失则阻塞财务章节
季报PDF + 附表政策证据与项目组合季度截止提醒政策资料缺失则仅给缺口清单
