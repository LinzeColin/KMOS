## 一、怎么跑

改完机器平面，按顺序跑这三条，让人类平面刷新并过关：

```bash
python3 machine/tools/render_human.py            # 先渲染人类平面
python3 machine/tools/check_doc_budget.py        # 体积、中文、纯净三道门
python3 machine/tools/check_blocker_stop.py      # 阻塞重审门
```

## 三、报错了怎么办

| 遇到什么 | 为什么 | 怎么解决 |
|---|---|---|
| 必需输入缺失 | 运行上下文、来源范围、公司事实或授权不足 | 停止并向负责人列出缺失字段；只有本次逐字段授权才能降级 |
| 来源覆盖下降 | 访问失败、登录门、验证码、附件失败或来源变更 | 显式标记覆盖降级并保留失败来源，不把无结果解释为无机会 |
| 企业协作平台不可读 | 命令不存在、未登录、管理员未授权或目标绑定冲突 | 停止真实读取，重新检查命令、状态、授权和精确目标 |
